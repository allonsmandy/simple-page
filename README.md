Este é um projetinho simples de front no qual foi utilizado na pratica o sass, gulp, BEM e npm scripts >u<

```
- npm install

- npm run gulp clean -> Limpa toda a pasta dist
- npm run gulp dist -> cria a pasta dist já bem performatica pronta para produção
- npm run gulp listen -> vai ficar monitorando os arquivos 
- npm run gulp server --> abre o servidorzinho para ver as alterações em real time :3
```
